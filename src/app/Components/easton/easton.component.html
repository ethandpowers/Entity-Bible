User Guide

The Easton's Bible Dictionary was originally published by Matthew George Easton in 1893. It has been updated through the years, and I am not certain as to the exact version used for the data in this project, which was compiled by the folks at the theographic project. To search for an entry in the Dictionary, simply select one of the following options from the drop-down menu above:

  • Term Label - The name of the entry in the dictionary. There may be multiple entries/definitions with the same term label. This is the recommended way to search.
  • Dictionary Lookup - A unique name similar to the term label, but only matches one entry/definition.
  • Term ID - A string of characters that uniquely identifies the term in the dictionary.
  • Definition ID - A string of characters that uniquely identifies the definition in the dictionary.
  • Index - An integer that represent the order of entry into the dictionary. Index 1 corresponds to the first entry in the dictionary.

Once you have selected one of the above options, enter the value you want to search for into the "Search Value" box to the right of the drop-down menu. To execute the search, press enter or click the search icon.

Note: While the search function can sometimes correct capitalization, it is recommended that if you don't get any results, that you try again with different capitalization.
