User Guide

The goal of the Entity Search Tool is to provice a central source of information for researching biblical entities such as people, places, etc. The data for this tool is actively being compiled, but is still incomplete. If something doesn't show up here, it is recommended that you try searching the Dictionary Tool. In the list below, you can see some of the different search parameters that are not self-explanatory:

  1. Person
    • Display Title - A more descriptive name, ex. "John the Baptist"
    • Also Called - A list of other names used to describe the person.
    • Relatives - Display a person's relatives of the specified type.
  2. Place
    • Aliases - Other names by which the place is known. This does not include the primary name.
    • People Born/Died - Searches for the place where the specified person died.

Note: The Entity Search Tool does not change the capitalization used in the search, since it can vary across different search types, attributes, and objects.
